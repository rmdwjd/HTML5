form태그내에 submit버튼 클릭시 해당 form 안의 작성된
사용자가 입력한 값들을 서버로 전송하면 요청하는 역할 수행

form action="" method=""

* action 속성 : 해당 form에 사용자가 입력한 값을 전달하면서 요청할 서버의 경로 제시
* method 속성 : 요청 전송 방식을 지정하는 속성(get : 기본값 | post )
> get 방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식
> post 방식 : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식
form내의 submit 버튼 클릭시 action지정한 서버로 사용자가 입력한 값들이
넘어가며서 요청됨(사용자가 넣은 값은 key=value 세트로 전달)

~~search.jsp?keyword=html5

라디오버튼과 체크박스 관련 태그

라디오버튼(type="radio")

- 여러개 값들 중 오로지 한개만 선택가능
- 서버로 value값이 전송
- name 속성의 값이 같아야 한개만 선택할 수 있다

성별 : 선택안함   남자   여자

성별 2 : 선택안함   남자   여자


체크박스(type="checkbox")

- 제시한 여러개의 값들 중 여러개를 선택할 수 있음 - 서버로 value의 값이 전송됨
